Proyek ini berisi implementasi setup protokol komunikasi antara transmitter (Tx) dan receiver (Rx) menggunakan dua board ESP Heltec LoRa WiFi. 
Sistem dirancang untuk mengirim dan menerima data secara nirkabel menggunakan teknologi LoRa dengan konfigurasi parameter yang sesuai. 
Pengujian dilakukan pada berbagai jarak dan kondisi lingkungan indoor untuk mengevaluasi performa komunikasi. 
Hasil menunjukkan bahwa komunikasi berjalan dengan baik, bahkan saat terhalang tembok dan perbedaan lantai, 
sehingga cocok untuk aplikasi IoT jarak menengah hingga jauh.
